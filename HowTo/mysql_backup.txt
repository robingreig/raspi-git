to backup a database:
mysqldump -urobin -pMicr0s0ft makerspace > /path/to/file.sql

to backup & zip a database:
mysqldump -urobin -pMicr0s0ft makerspace > /path/to/file.sql | gzip > /home/robin/mariadb/makerspace_backup.`date+%F`.sql.gz

to restore a database:
# you have to have that database in the new mariadb (CREATE DATABASE IF NOT EXISTS makerspace)
mysql -urobin -pMicr0s0ft makerspace < /path/to/file.sql

to restore a zipped database:
gunzip < zipfile.sql.gz | mysql -urobin -pMicr0s0ft makerspace
